# open-codex-computer-use

## 简介

一个面向 Agent 协作开发的基础仓库模板，可以用来启动任何你想做的产品或服务。

## 快速开始

直接 clone 或 fork 这个仓库，然后把它作为新项目的起点，继续往里搭你真正要做的产品、服务或平台能力即可。
